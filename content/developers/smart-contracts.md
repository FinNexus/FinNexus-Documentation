# Ethereum and Wanchain Smart Contracts

The following are the Ethereum and Wanchain smart contract addresses and the corresponding file names to the contract files found on [our Github](https://github.com/FinNexus/FinnexusOptionsV1.0/tree/master/contracts). As the smart contracts are the same for both chains, in order to get the ABIs and other related information for both, you may use the [information from Etherscan](https://etherscan.io/address/0x919a35a4f40c479b3319e3c3a2484893c06fd7de#code) (WanScan still does not support features such as ABI look up).
